# Excel Estimating Mini-Repo (Robust Assemblies)

## Download (recommended)
➡️ **Download the workbook:**  
assets/Excel_EstimMiniRepo_Robust_EN_LocaleSafe.xlsx

## How to use (60 seconds)
1) Open the workbook in Excel
2) Go to **Inputs** and set Overhead / Profit / Tax
3) Go to **Assembly_Builder**
   - Pick **Assembly Code** (B3)
   - Set **Multiplier** (B4)
4) Copy **Item_Code** + **Qty** into **Estimate** (columns A and D)
5) Review totals at the bottom of **Estimate**

## What “Multiplier” means
**Qty = Factor_Qty × Multiplier**
- 1.00 = base package
- 1.25 = +25% scope
- 0.80 = -20% scope
